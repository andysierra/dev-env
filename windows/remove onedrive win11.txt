Method 2: Using Registry Editor

Press Windows Key + R.
Type regedit and click on OK.
In the newly opened window, open the following registry hive:
HKEY_CLASSES_ROOT\CLSID\{018D5C66-4533-4307-9B53-224DE2ED1FE6}
Double-click on a string value named System.IsPinnedToNameSpaceTree on the right-hand side panel.
Type 0 and click on OK.
